Customize Weights

Adjust how much each criterion matters. Changes apply immediately to live scores and rankings.

How this works: Each criterion has a weight from 0 (excluded) to 5 (maximum priority). The default weights reflect the tiered system: T1 = 3 (most important), T2 = 2, T3 = 1. Pushing a slider to 0 excludes that criterion entirely.

Cap at: Use this when a criterion has a "good enough" threshold. For example, capping Progressive at 4 means any city rated 4 or higher is treated equally — a 5-rated city gets no extra credit. Useful when "above threshold X is enough."

Note on median house price: The dollar figure shown per row is informational only and not currently part of the score. Only the "Land for build" rating (rural buildable parcel cost) feeds into the live score. The median home price is the typical urban home value, while landScore reflects rural land for a custom build — these can diverge significantly.
